# KBOSearch 코드베이스 개선 체크리스트

> 작성일: 2026-04-27
> 목적: 코드베이스 개선점을 우선순위별로 정리하고, 작업 진행 시 체크리스트로 관리함.

---

## 진행 원칙

- 작업은 우선순위 순서대로 진행함.
- 각 항목은 구현 완료 후 체크함.
- 기능 동작을 바꾸는 개선은 가능한 한 작은 단위로 나눠 검증함.
- 크롤링/분석 결과가 바뀔 수 있는 작업은 수동 실행 또는 테스트 스크립트로 확인함.
- 기존 사용자 변경사항은 되돌리지 않고, 필요한 범위만 수정함.

    return f"{BASE_URL}/Schedule/Schedule.aspx?gameDate={game_date}"

def game_center_url(game_date: str, game_id: str, section: str) -> str:
    return (
        f"{BASE_URL}/Schedule/GameCenter/Main.aspx"
